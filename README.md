## Featured Projects

### [Epiderma Shield](https://github.com/adarsh288/Epiderma-Shield)
> AI-powered skin disease detection web app

- 6-layer CNN trained on 640+ dermatoscopic images — **98.44% accuracy**
- Flask REST API serving real-time predictions, validated via Postman
- React.js frontend for seamless user interaction
- Backed by a published research paper — [IJRASET 2025](https://doi.org/10.22214/ijraset.2025.72462)

`TensorFlow` `Keras` `Flask` `React.js` `Scikit-learn` `Postman`

---

### [Plant Leaf Disease Detection](https://github.com/adarsh288/plant-leaf-disease-detection)
> Deep learning pipeline for early crop disease identification

- CNN trained on 2,152 potato leaf images — **97.26% accuracy**
- Data augmentation and hyperparameter tuning for robust generalisation
- Helps farmers detect Early Blight and Late Blight before crop loss occurs

`TensorFlow` `Keras` `Scikit-learn` `NumPy` `Saturn Cloud`

---


### 🏏 [T20 Cricket Performance Analysis](https://github.com/adarsh288/T-20-Cricket-Performance-Analysis)
> Data-driven approach to selecting the best XI players from T20 cricket statistics

- Scraped real-world T20 match data using Python and web scraping techniques
- Cleaned and transformed raw data using Pandas for analysis-ready datasets
- Built an interactive Power BI dashboard to rank and filter players across key metrics — batting average, strike rate, bowling economy, and more
- Final output: a data-backed selection of the **Top 11 T20 players** based on performance statistics

`Python` `Pandas` `Web Scraping` `Power BI` `Data Analytics`
